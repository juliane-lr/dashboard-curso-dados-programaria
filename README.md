# 📊 Dashboard Interativo: Análise de Diversidade no Cenário de Dados Brasileiro (2022)

Este projeto consiste em um dashboard interativo desenvolvido no **Looker Studio** focado em analisar a diversidade de gênero, etnia, escolaridade e distribuição regional dos profissionais da área de dados no Brasil. 

O painel foi construído durante o **Curso de Análise de Dados da PrograMaria**, utilizando como base os dados reais da renomada pesquisa **State of Data Science 2022**, realizada pela comunidade **Data Hackers**.

---

## 🎯 Objetivo do Projeto

O mercado de tecnologia e dados cresce aceleradamente, mas a sub-representação de grupos minorizados ainda é um desafio real. O principal objetivo deste projeto foi mapear e cruzar dados demográficos com métricas de carreira (como senioridade e média salarial) para dar visibilidade ao panorama de diversidade do setor, transformando dados brutos em insights visuais acionáveis.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **Python & Google Colab** (Extração, limpeza, tratamento e estruturação inicial da base de dados)
* **Looker Studio** (Criação do dashboard, modelagem final, design de interface e cruzamento de visuais)
* **Dataset:** Base de dados do *State of Data Science 2022 (Data Hackers)*, contendo mais de 4.000 respostas de profissionais de dados de todo o Brasil.
---

## 📈 Estrutura do Dashboard e Principais Insights

O relatório é dividido de forma lógica em 3 páginas principais de análise profunda:

### 1. Visão Geral (Análise de Diversidade 2022)
* **Métricas Principais (KPIs):** Visão consolidada com a quantidade total de respondentes (4.271), volume do gênero feminino (1.056), quantidade de pessoas não-brancas (1.527), profissionais PCD (54) e a descentralização geográfica (2.411 profissionais fora do estado de SP).
* **Distribuição Geográfica:** Um mapa de calor interativo destacando a forte concentração de profissionais na região Sudeste, facilitando a identificação de polos emergentes em outros estados.
* **Gênero x Etnia:** Gráfico de colunas empilhadas demonstrando a proporção de cada grupo demográfico na base de dados.

### 2. Análise de Gênero
* **Recorte de Escolaridade e Senioridade:** Gráficos que evidenciam como a proporção de mulheres varia conforme o nível de ensino e o cargo ocupado, identificando o funil de representatividade em cargos de liderança (*Pessoa Gestora*).
* **Média Salarial por Gênero:** Um insight crítico que demonstra visualmente a disparidade salarial média entre homens (R$ 10,4 mil) e mulheres (R$ 8,5 mil) no mesmo mercado de atuação.
* **Filtro Interativo:** Possibilidade de segmentar toda a página por *Faixa Etária* para análises mais granulares.

### 3. Análise de Etnia
* **Etnia x Região do Brasil:** Gráfico de barras 100% empilhadas que correlaciona a autodeclaração de cor/raça com as diferentes regiões do país.
* **Média Salarial por Etnia:** Comparativo direto das remunerações médias praticadas no mercado para cada grupo ético.
* **Acessibilidade:** Inclusão de um botão funcional para o usuário baixar o relatório completo diretamente pela plataforma.
